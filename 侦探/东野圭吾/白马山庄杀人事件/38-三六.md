    三六 

   菜穗子关上房门时，中村仍然一脸茫然地坐在椅子上。菜穗子穿过走廊，听到身后的房间传来某种东西碰撞房门的闷响。中村应该没有勇气丢酒杯，想必是丢了枕头之类的东西吧。不管他丢的是甚么，菜穗子都不想理会愚蠢的男人。

   老板的气色虽然不太好，但是他站在吧台里面亲切地陪着菜穗子说话。对于菜穗子提的问题，老板也认真地回答。

   “您是说‘启程’的房间的命名由来吗？这问题很难回答呢。”

   “您不知道由来吗？”

   “坦白说，我不知道。英国的朋友把这里卖给我的时候，那间房间就已经被这样命名了。确实如您所说，雕刻在壁饰上的歌词和‘启程’似乎没有任何关联。”

   “‘启程’这两个字是老板您翻译的吧？原文是……”

   “原文是 start，所以也可以翻成‘出发’。可是，我想到这里是一座度假山庄，用‘启程’的意境似乎比较好。”

   “start……原来是这样啊，原来是 start。”

   菜穗子刚刚因为被中村催促着，所以没能够确认到房门上的门牌。

   菜穗子在心里暗自念出：

   白色地面上，

   有黑色种子；

   想解开谜题，

   就得勤用功。

   “谜题”这个词轻轻刺激着菜穗子的脑部，为何这首童谣是“start”呢？

   “该不会……”菜穗子不禁发出声音。

   专心煮着咖啡的老板似乎没听清楚菜穗子说的话，他问道：“甚么事？”

   菜穗子急忙摇摇头回答：“没事。”

   该不会这首童谣是解开暗号的第一步吧？——这是菜穗子前一刻想到的事。

   或许 start 不应该翻译成“启程”或“出发”，而是翻译成好比说——“开始”呢？而且，“想解开谜题，就得勤用功”这一小句歌词的意境，很符合解开暗号的序曲。

   “老板，谢谢招待！”太过兴奋的菜穗子忘了自己还没喝任何饮料，便朝老板喊道，随后匆忙地跑回房间。她感觉到自己全身在发热。

   菜穗子回到房间、锁上门锁后，便取出高濑画的位置图。她再次确认每间房间的配置，发现事情正如她所想，她兴奋地点了点头。

   除去“伦敦铁桥与年纪一大把的鹅妈妈”的房间不看，“开始”的房间——菜穗子已认定 start 只能这么翻译——就是山庄里位置最旁边的房间。况且，“伦敦铁桥与年纪一大把的鹅妈妈”并不在同一栋建筑里。

   菜穗子想起芝浦夫妇提过，公一曾表示解开暗号的诀窍在于按照各房间的顺序阅读童谣。也就是说，应该以“开始”的房间为起点，按照顺序一路阅读下去就行了。这么一来的话，下一首童谣就会是……

   菜穗子的视线正好落在写着“圣保罗”的文字上时，出入口的房门传来摇晃门把的声音，应该是真琴回来了。菜穗子一打开门，真琴随即用大拇指与食指做出圆圈的手势。

   “你的表情看来是有所收获的样子。”

   “你的气色也不错嘛！”真琴说罢，瞥了走廊一眼后，关上房门。

   “我有事想告诉你。”

   “那就先听你说吧。”

   两人中间夹着桌子，面对面而坐。

   菜穗子向真琴说明了“启程”应该翻译成“开始”的推测，这首童谣应该就是解开暗号的第一首歌，歌词里也提到黑色种子。

   真琴一边看着菜穗子写下来的《开始》的歌词，一边喃喃说：“这方向应该正确，问题是这个黑色种子指的是甚么？看来有必要再拜访一次医生夫妇的房间。”

   “我也这么想。”菜穗子表示赞同。

   “对了，真琴，你的收获呢？你应该问出很多事情了吧？”

   “是啊。”真琴咧嘴一笑，接着从牛仔裤口袋里掏出一张纸，并在菜穗子面前摊开来。上面的字迹棱角分明、非常男性化，写得有点潦草。菜穗子知道那是真琴独特的笔迹。

   “两年前坠落死亡的人叫作川崎一夫，他以前在新宿经营珠宝店，年约五十岁。当时他不是第一次来这里投宿，听说在那之前半年的夏天，他曾来过一次。他是在第二天的晚上从石桥坠落，当时的判断是因为脚滑所以摔落。”

   “也就是说，没有像这次一样的手法啰？”

   “事到如今当然没办法确认了，不过如果有，警察应该不至于没有发现他杀的迹象。”

   “说得也是。”

   “主厨对这个人的印象是沉默寡言、有些阴沉，听说他当时几乎没有和其它客人交谈。当时的客人到现在还会来这里的包括医生夫妇、芝浦夫妇，还有江波。可是，那时大家的同伴意识没那么强，所以大家对事件都表现得漠不关心。不过，听说这事件背后还有一些故事，是主厨在参加川崎的葬礼时，从川崎的亲戚那里听来的。”

   “甚么背后的故事？”

   菜穗子记得有人曾经告诉过她举办葬礼的时候，有关往生者生前的传言会像百货公司在举办大甩卖一样，被贱价出售。

   “在那之前，得先说一件重要的事。”真琴不是故意要吊人胃口，她用慎重的语气说道：“主厨很少把这件事告诉别人，我想这也是因为很少有人问他的关系吧。主厨说他一直极力避免谈到这事情，但你猜，最后一个问主厨的人是谁？”

   “嗯……”菜穗子开始思考。真琴会这么问，一定有她的用意。菜穗子抬起头。“该不会是……我哥哥？”

   “答对了！”真琴说：“公一也知道这件事情，这个事实值得重视。也就是说，我们和公一几乎是循着同样的路径在走。”

   “你的意思是哥哥企图解开暗号的事情，和两年前的事件并非毫无关联吗？”

   “就是这么回事，而说到这关键的背后故事。”真琴说着，在菜穗子面前竖起三根手指头。“一共有三个。”

   “三个？”

   “没错，不过听说主厨通常只会讲两个。他的理由我等会儿再说明，就先讲前面两个故事吧。第一个故事是在川崎的亲戚之中，有谣言说那不是意外，而是自杀，听说那些亲戚都确信川崎是自杀死的。”
